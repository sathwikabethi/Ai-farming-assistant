# 🌾 Rythu Mitra – AI Assistant for Farmers

Rythu Mitra is an AI-powered smart farming assistant built using **Python, Streamlit, and Google Gemini AI**.
It helps farmers make **data-driven agricultural decisions** by providing crop disease detection, weather advisory, soil analysis, expense tracking, and more in one unified platform.

The system is designed to improve **farm productivity, reduce crop losses, and assist farmers with real-time intelligent insights**.

---

# 🚀 Features

### 🌿 Crop & Disease Detection

Upload crop images to detect plant diseases and receive treatment recommendations using **AI image analysis**.

### 🤖 AI Farming Chatbot

An intelligent chatbot powered by **Google Gemini AI** that answers farming-related questions.

### ☔ Weather Advisory

Provides real-time weather information and farming recommendations based on forecast data.

### 🧪 Soil & Fertilizer Advice

Suggests fertilizer plans based on:

* Soil pH
* Nitrogen
* Phosphorus
* Potassium
* Crop type

### 📈 Market Price Analysis

Displays mandi price trends for major crops to help farmers decide **when and where to sell crops**.

### 💰 Farm Expense Tracker

Track farm income and expenses with:

* Transaction logging
* Expense categorization
* Profit/loss analysis

### 🗓️ Crop Calendar

Generates a **complete crop growth schedule** including irrigation, fertilizer stages, pest monitoring, and harvesting timeline.

### 💧 Water Management

Calculates irrigation schedules based on:

* Crop type
* Soil type
* Farm area
* Weather
